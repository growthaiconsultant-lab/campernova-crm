# Estructura profesional para anuncios de campers

El anuncio se entrega siempre en este orden exacto. **Nunca mezclar secciones.**

## 1. Título optimizado

Generar siempre 3 títulos. Estructura recomendada:

> Marca + Modelo · motor o cambio · plazas · extras principales

Ejemplo:

> Adria Matrix 650 TR | Automática | 5 plazas | Litio + Solar + Inversor

## 2. Descripción inicial

Breve resumen del vehículo (2-3 frases). Tono profesional y cercano.

## 3. Datos del vehículo

- Marca
- Modelo
- Motor
- Cambio
- Plazas

## 4. Medidas

- Longitud
- Anchura
- Altura

## 5. Equipamiento destacado

- Instalación solar
- Baterías
- Inversor
- Calefacción
- Aire acondicionado

## 6. Sistema eléctrico

- Baterías
- Inversor
- Placas solares
- Cargador

## 7. Sistema de agua

- Depósito de aguas limpias
- Depósito de aguas grises

## 8. Distribución vivienda

- Tipo de camas
- Zona cocina
- Zona salón

## 9. Llamada a la acción

(Ver `templates/cta.md`)

## 10. Condiciones de venta

(Ver `templates/sales-conditions.md` — bloque fijo, nunca dividir)

## Reglas anti-alucinación

- Nunca inventes datos técnicos si no se ven claramente en imágenes o no los proporciona el usuario.
- Si no tienes un dato concreto, **omite la línea** en lugar de poner "no especificado".
- Si las medidas coinciden con una versión L/H estándar, puedes indicar la equivalencia. Si no, usar solo las medidas reales.
